# random_walk_navigation
Pacote de navegação ROS do tipo random walk para AMRs, esse pacote foi criado, em parceria com a empresa Selettra Automação e Robótica, para que o AMR possa navegar em ambientes fechados transportando dispositivos de esterilização de ambientes como por exemplo lâmpadas que emitem raios UV do tipo C ou de ozônio. Dispensando a persença humana no ambiente garantindo sua segurança.
